Using SwiftMark is remarkably easy.


You start by adding the package to your `Package.swift`:



```swift
.Package(url: "https://github.com/calebkleveter/SwiftMark.git", majorVersion: 1,
 minor: 2)
```

Then you import SwiftMark, create a renderer and render the Markdown to HTML:



```swift
import SwiftMark

let mdRender = MarkdownRenderer
let html = mdRender.render(markDownText)
```
