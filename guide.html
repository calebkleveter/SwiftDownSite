Using SwiftDown is remarkably easy.


You start by adding the package to your Package.swift:



```swift
.Package(url: "https://github.com/calebkleveter/SwiftDown.git", majorVersion: 0,
 minor: 1)
```

Then you import SwiftDown, create a renderer and render the Markdown to HTML



```swift
import SwiftDown

let mdRender = MarkdownRenderer
let html = mdRender.render(markDownText)
```
